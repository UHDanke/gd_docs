# Options

Changes misc level and players options.

Each option can be set On, Off or left unchecked.

## Streak Additive

Makes the player's trail blending when enabled and solid when disabled.

On by default.

## Hide Ground

Hides the level's ground when enabled.

Off by default.
 
## Hide Middleground

Hides the level's middleground when enabled.

Off by default, but MG is not visible unless set by Change Middleground.

## Hide P1

Hides player 1 when enabled.

Off by default.

## Hide P2

Hides player 2 when enabled.

Off by default.

## Disable P1 Controls
Prevents inputs from controlling player 1 when enabled.

Player 1 will not move or jump in response to any input. Also affects most triggers and interactibles that require player input, with the exception of Touch trigger.

Off by default.

## Disable P2 Controls

Prevents inputs from controlling player 2 when enabled.

Player 2 will not move or jump in response to any input. Also affects most triggers and interactibles that require player input, with the exception of Touch trigger.

Off by default.

## Unlink Dual Gravity

Prevents players from synchronizing gravity when enabled if one of the players interacts with the following objects:
- Gravity portals
- Blue pads
- Blue Orbs
- Green Orbs
- Gravity Dash Orbs

Toggling on 2 Player mode also unlinks gravity.

Off by default.

## Hide Attempts

Hides the attempts text at the start of the level when enabled.

Overriden by the **Hide Attempts** game option.

Off by default.

## Audio On Death

Off by default.

## No Death SFX

Makes the on death SFX will not play when enabled.

Off by default.

## Edit Respawn Time

Sets the **Respawn Time** when enabled and resets it to its default value when disabled.

Off by default.

**Respawn Time** is limited between 1.00 and 10.00 seconds, even if you are using an option that reduces respawn time like **Fast Respawn**

## Boost Slide

Off by default.
